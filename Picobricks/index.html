PicoBricks Is A Fantastic Robotic Project Development Kit For STEM Educators And Makers Of All Levels. No Soldering And Wiring. There Is Unlimited Creativity!

What is PicoBricks?

PicoBricks is a Raspberry Pi Pico based electronic development board and software designed to speed up the process of learning robotic programming and make maker projects easy, ideal for building robot applications and Raspberry Pi robotics projects. It is programmable and perfect for STEM learning, offering block based coding capabilities. There are 12 removable modules on PicoBricks, making it a versatile and adaptable tool. This makes it a tool with unlimited possibilities for your projects, especially when integrated with Python programming.
